# 🚀 AI MVP Builder

AI MVP Builder is a multi-agent AI system that transforms a startup idea into a full-stack MVP architecture by simulating a software engineering team.

---

## 🧩 Problem

Building an MVP requires multiple roles — system design, backend development, and frontend setup — making it slow and complex for solo builders and early-stage founders.

---

## ⚡ Solution

AI MVP Builder automates this process using specialized AI agents that collaboratively generate a structured MVP architecture from a simple idea.

---

## 🛠️ Features

- 🧠 Planner Agent for system design & API planning  
- ⚙️ Backend Agent for FastAPI structure generation  
- 🎨 Frontend Agent for React architecture generation  
- 🔗 Multi-agent orchestration workflow  
- 📊 Streamlit interface for end-to-end interaction  

---

## 🔄 Workflow

User Idea → Planner Agent → Backend Agent → Frontend Agent → Generated MVP Architecture

---

## ⚙️ Tech Stack

- Python  
- Streamlit  
- Gemini API  
- FastAPI  
- React  
- Multi-Agent AI Workflow
